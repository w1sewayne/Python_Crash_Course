# 🐍 Python Chapter 7: 用户输入与 While 循环 (Input & While Loops) 速查表

> **核心概念**：如果说之前的程序是“独角戏”（按写好的剧本演完就结束），那这一章就是教会程序如何与人“对话”（Input），以及如何根据情况“一直工作下去”（While 循环）。

## 1. 用户输入 (User Input)

**生活类比**：`input()` 就像填表。程序暂停下来，把麦克风递给你，等你说话。

| 功能 | 英文关键字 | 代码示例 | 备注/易错点 |
| :--- | :--- | :--- | :--- |
| **获取输入** | `input()` | `name = input("Who are you? ")` | **⚠️ 陷阱**：无论你输入什么（哪怕是数字），Python 都默认它是**字符串 (String)**。 |
| **数值转换** | `int()` | `age = int(age_str)` | **必做动作**：如果你要拿输入的数字做数学运算或比较大小，必须先用它“剥壳”。 |
| **求模(余数)** | `%` (Modulo) | `4 % 3` 结果是 `1` | **判断奇偶的神器**：`if num % 2 == 0` (是偶数)。 |

## 2. While 循环控制 (Controlling Loops) [🔴重难点]

**生活类比**：`while` 循环就像“看门大爷”或者“玩游戏”。只要条件满足（比如你还有血量），游戏就一直继续；条件不满足（血量为0），游戏结束。

### 循环退出的四种姿势

| 方式 | 语法/关键字 | 适用场景 | 生活类比 |
| :--- | :--- | :--- | :--- |
| **条件测试** | `while x <= 5:` | 最简单的计数循环。 | **倒计时**：数到0就停。 |
| **标志变量** | `Flag` (如 `active`) | 程序有多个退出条件时（如游戏结束、玩家退出、系统故障）。 | **交通信号灯**：只要灯是绿的(`True`)车就一直走，变红(`False`)所有车都停。 |
| **立即退出** | `break` | 不管循环条件如何，立刻强行终止循环。 | **紧急出口**：发生火灾，直接推门跑路，不管原来的流程了。 |
| **跳过本次** | `continue` | 忽略余下的代码，直接回到开头进行下一次循环。 | **跳过坏苹果**：这一题不会做，跳过，直接做下一题（不交卷，只是跳过这一项）。 |

**💡 记忆口诀**：
* **Break** = **B**ye bye (彻底再见)
* **Continue** = **C**arry on (继续下一个)

## 3. 使用 While 处理列表与字典

**核心区别**：`for` 循环适合“遍历”（把所有东西看一遍），而 `while` 循环适合“搬运”或“修改”（只要列表里还有东西，就一直处理）。

| 操作 | 代码思路 | 典型应用 |
| :--- | :--- | :--- |
| **列表间移动** | `while unconfirmed:`<br>`  user = unconfirmed.pop()`<br>`  confirmed.append(user)` | **验证用户**：把未验证列表里的用户一个个弹出来，检查完扔进已验证列表。 |
| **删除特定值** | `while 'cat' in pets:`<br>`  pets.remove('cat')` | **批量清理**：`remove()` 一次只删一个，配合 `while` 可以把列表里所有的 'cat' 删得干干净净。 |
| **用户填充字典** | `while active:`<br>`  name = input(...)`<br>`  resp = input(...)`<br>`  answers[name] = resp` | **问卷调查**：不断询问“还有人要回答吗？”，收集所有人的名字和答案存进字典。 |

## 4. ⚠️ 常见坑点与防坑指南 (Troubleshooting)

* **无限循环 (Infinite Loop)**：
    * **现象**：程序疯狂刷屏，停不下来。
    * **原因**：忘记在循环里更新变量（比如忘记写 `x += 1`），导致条件永远为 `True`。
    * **急救**：按 **`Ctrl + C`** 强行终止程序！
* **Input 的提示词**：
    * 如果提示词太长，可以先赋值给一个变量 `prompt`，再 `input(prompt)`，代码会更整洁。
